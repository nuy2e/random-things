# PDF Merger Utility

A simple, reusable Python script to merge multiple PDF files into a single document using the pypdf library.

## Features
It merges any number of PDF files in the specified order.

## Prerequisites

Ensure you have Python installed. You will need to install the pypdf library to run this script: pip install pypdf

## Usage
1. Place your PDF files in the same directory as merger.py.
2. Update the files_to_merge list in the if __name__ == "__main__": block of the script with your desired filenames.
3. Run the script: pdf_merger.py
